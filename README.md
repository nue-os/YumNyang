## 📝 소개
**먹었냥🐾** 프로젝트는 사랑하는 반려동물의 건강하고 행복한 식생활을 위해, 
반려인들이 직접 만드는 특별한 레시피를 공유하고 '반려동물 집밥원료 API'를 통해 정확한 영양 정보까지 확인할 수 있는 React 기반 웹 서비스입니다.
